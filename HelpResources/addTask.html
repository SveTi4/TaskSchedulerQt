Добавление задачи


Для создания задачи необходимо нажать на кнопку со значком "+", после чего появится окно, куда нудно ввести:

  • Название задачи (не может быть пустым)
  • Описание задачи
  • Дату, на которую необходимо запланировать задачу

Для завершения добавления нажмите на кнопку "Добавить" и задача сразу отобразится в основном окне в указанную дату
